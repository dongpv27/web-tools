
# KẾ HOẠCH XÂY DỰNG WEBSITE TOOL (PHẦN 1)
## Mục tiêu
Xây dựng website chứa ~100 micro tools, tối ưu SEO long-tail, code chủ yếu client-side để giảm chi phí server.
Website có thể tạo nhanh bằng AI (Claude/Codex).
